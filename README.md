# ProjBirdie

WebApp to Visualize Data on Birds in 3D Environment to bring a new and different approach on the problem of **Avian Extinction**.

## Made with :heart: and these Tech's.

- Angular Version 9.0
- GraphQL
- Three JS
- MongoDB
- Firebase
- Express

## Commands

Build Server

    make build-server

Build Website

    make build-website
